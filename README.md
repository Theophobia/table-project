Directory "docs" contains documentation generated by doxygen. It can be viewed by opening "/docs/html/index.html".

Directory "table-project" contains header and implementation files for the project.

Directory "table-project-tests" contains unit tests, the test entry point and the unit testing library (doctest).
For more information, https://github.com/onqtam/doctest

Tools used:
- clang 11.1.0
- cmake 3.16.5
- doctest 2.4.6
- doxygen 1.9.1
- clion 2020.1
